# Paul, The Octopus

![./img/paul_octopus.png]

Há 8 anos atrás, o "Polvo Paul" fez história na África do Sul ao acertar mais de 85% das suas previsões de vencedores na Copa do Mundo de 2010. 

## Sobre este repositório

Este repositório é um conjunto de scripts R que usa a Google Cloud Platform para tentar prever os resultados dos jogos da copa do mundo da Rússia.

## Requisitos

O código R vai levantar instâncias para executar simulações e acessar o BigQuery e Storage restritos de um projeto interno da CI&T, vc precisa ter acesso à esse projeto a fim de conseguir reproduzir a sua execução.
