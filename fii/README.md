# FII Dashboard

Conjunto de scripts para acessar dados de uma carteira de fundos de investimento imobiliários registrados em um Google Spreadsheet, fazer scrapt de dados de proventos do site http://www.fii.com.br desses fundos e obter as cotações via pacote GetBatchSymbol a fim de calcular e apurar os rendimentos dessa carteira.

A idéia é montar um Dash automatizado partir dessas informações.

Obviamente vc não conseguirá a parte que acessa a carteira, pois está restrita ao meu Drive, mas pode olhar os códigos (ainda em draft) para obtenção das cotações e scrap do proventos.
